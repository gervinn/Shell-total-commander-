# 18. Shell (total commander)  
Оболонка повинна вміти виконувати основні дії в системі – перегляд файлів та папок у файловій системі, перемикання між дисками, копіювання, видалення, переміщення об'єктів, пошук.

---

## Запуск серверу: dotnet run --project ServerApp/ServerApp.csproj
## Запуск програми dotnet run --project ShellTotalCommander1/ShellTotalCommander1.csproj
Запуски виконувати в різних терміналах
